# О расширении

Jira Helper — это браузерное расширение, которое добавляет в Jira полезные функции для управления задачами: WIP-лимиты, визуализацию, диаграммы Ганта, аналитику и многое другое.

Расширение работает с **Jira Server** и **Jira Data Center**. Jira Cloud не поддерживается.

**Поддерживаемые браузеры:** Chrome, Edge, Yandex и другие браузеры на Chromium, а также Firefox. Safari не поддерживается.

Все функции опциональны и настраиваются отдельно для каждой доски.

> **Впервые с Jira Helper?** Начните с [быстрого старта](/docs/getting-started/quick-start).
