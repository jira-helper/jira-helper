# Лимиты WIP по ячейкам доски

| | |
|---|---|
| Где настраивается | Настройки доски → Колонки → **«WIP-лимиты на ячейки»** |
| Где видно | Доска (детальный вид) |
| Настройки действуют | Для всей команды |

## Цель

Задать единый лимит незавершённой работы для выбранного блока ячеек доски (пересечений колонок и swimlane). Область выделяется контуром, на отдельных ячейках можно включить счётчики, а при превышении лимита ячейки подсвечиваются. Диапазоны можно пометить как отключённые — для визуальной группировки без активного WIP-контроля.

Ячейка Team Backend × In Progress имеет WIP-лимит 3 — в ней сейчас 4 задачи (перегруз, пунктирная рамка, красный бейдж 4/3).

## Как настроить

### Где найти настройки

1. Откройте **«Настройки доски»**.
2. Перейдите на вкладку **«Колонки»**.
3. Нажмите **«WIP-лимиты на ячейки»**.

### Как настроить

Для каждого диапазона доступны следующие параметры:

- **«Имя»** — название диапазона.
- **Лимит** — числовой порог для всех задач внутри диапазона.
- **«Отключён»** — пометить диапазон как визуально отключённый (штриховка, без активного поведения WIP).
- **«Типы задач»** — сузьте, какие типы задач учитываются в диапазоне.
- **«Ячейки»** — таблица пар swimlane–колонка, определяющих границы диапазона. Для каждой ячейки можно включить или выключить индикатор-бейдж.

Нажмите **«Сохранить»**, чтобы сохранить, или **«Отмена»**, чтобы отменить.

## Как использовать

- Закрытый блок ячеек обводится **синей пунктирной рамкой**, чтобы форма диапазона была очевидна.
- На ячейках с включённым индикатором показывается бейдж **текущее / лимит** с цветом: **зелёный** (ниже), **жёлтый** (на лимите), **красный** (выше).
- При превышении лимита все ячейки диапазона получают **полупрозрачный красный** фон.
- **Отключённые** диапазоны отображаются со **штриховкой** и не проявляют активного WIP-поведения — удобно для визуальной группировки без ограничений.
- Все задачи внутри ячеек диапазона суммируются в **один общий счётчик**.

## Сценарии использования

- «Хочу ограничить пятью задачами всё, что находится в колонке In Progress и swimlane Frontend.»
- «Хочу визуально сгруппировать блок ячеек без ограничений, пометив его как отключённый.»
- «Хочу считать только баги внутри конкретного диапазона ячеек.»
- «Хочу разные диапазоны ячеек с независимыми лимитами для разных частей доски.»
