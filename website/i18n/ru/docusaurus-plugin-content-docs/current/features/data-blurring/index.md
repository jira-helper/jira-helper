# Размытие данных

## Обзор

Режим конфиденциальности и презентации. Скрывает текст и размывает изображения на всех страницах Jira — досках, представлениях задач, поиске, бэклоге, настройках — чтобы вы могли демонстрировать экран или записывать демо, не раскрывая заголовки задач, ключи, имена или описания. Макет остаётся неизменным; содержимое нечитаемо.

**Зачем:** Демонстрация экрана во время встреч, публичных демо или записи обучающих видео часто требует скрытия конфиденциальных данных Jira. Размытие данных включается мгновенно из контекстного меню — не нужно открывать отдельный инструмент, применять фильтры или редактировать скриншоты.

**Где:** Контекстное меню по правому клику на любой странице Jira → **Blur secret data** (Размыть конфиденциальные данные). Метка «Blur secret data» относится к визуальному скрытию (CSS-размытие), а не к шифрованию — данные скрыты от просмотра, но не защищены криптографически. Переключатель привязан к профилю браузера и сохраняется между вкладками и сессиями.

**Как это работает:**
- При включении текстовое содержимое на страницах Jira отображается как нечитаемое размытие (CSS blur + text-indent)
- Изображения (аватары, скриншоты, вложения) размываются через CSS-фильтр
- Макет страницы, позиции карточек, дорожки, колонки и структура сетки остаются неизменными
- Переключатель — это простой флажок в контекстном меню расширения браузера — панель настроек Jira не нужна

## Что размывается, а что остаётся видимым

| Элемент | Размывается | Примечания |
|---|---|---|
| Ключи задач (например, PROJ-1234) | Да | Отображаются как размытый текст |
| Заголовки задач | Да | Полный заголовок размыт |
| Описания | Да | Весь текст размыт |
| Имена пользователей / исполнители | Да | Аватары также размыты |
| Значения пользовательских полей | Да | Любой текст в полях размыт |
| Превью вложений | Да | Изображения размыты через CSS-фильтр |
| Аватары / значки пользователей | Да | Применён CSS-фильтр размытия |
| Заголовки колонок (например, «К выполнению») | Нет | Названия статусов рабочего процесса остаются читаемыми |
| Названия дорожек | Нет | Метки структуры доски сохраняются |
| Позиции карточек / макет | Нет | Макет сетки не изменяется |
| Кнопки, значки, хром интерфейса | Нет | Элементы управления интерфейсом Jira остаются видимыми |
| Текст быстрых фильтров | Нет | Метки фильтров остаются читаемыми |

**Примечание:** Текстовые элементы SVG и содержимое, отрисованное на canvas, не могут быть размыты через CSS и остаются видимыми. Эффект размытия применяется только к стандартным HTML-элементам текста и изображений.

## Первое знакомство

1. Установите расширение и перейдите на любую доску или страницу задачи Jira.
2. **Кликните правой кнопкой мыши** в любом месте страницы Jira.
3. В контекстном меню найдите запись расширения **Blur secret data** (в подменю Jira Helper в браузерах на Chromium; на верхнем уровне в Firefox).
4. **Кликните** по записи, чтобы включить её (рядом появится галочка).
5. Весь текст мгновенно размывается — макет остаётся прежним.
6. Кликните снова, чтобы выключить и восстановить читаемость.

## Использование

### Включение/выключение через контекстное меню по правому клику → «Blur secret data»

**Цель:** Быстро включить или отключить визуальное скрытие без открытия настроек Jira.

1. На любой странице Jira, где расширение активно, **кликните правой кнопкой мыши** по странице (не по ссылке или изображению — кликните по пустому месту или фону страницы).
2. Найдите запись расширения **Blur secret data** в контекстном меню.
   - **Chrome/Edge:** Ищите в подменю **Jira Helper** контекстного меню
   - **Firefox:** Ищите на верхнем уровне контекстного меню
3. Кликните как по флажку: **отмечено** = размытие включено, **не отмечено** = размытие выключено.
4. Переключатель **сохраняется между вкладками и перезапусками браузера** — будучи включённым, он остаётся включённым, пока вы его не выключите. Он хранится в `localStorage` (см. [Настройки: Локальное хранилище](/docs/settings#local-storage-personal)).
5. Отдельные настройки доски Jira не нужны; переключатель находится исключительно в контекстном меню расширения браузера.

### Презентация с включённым размытием: текст скрыт, изображения размыты, макет сохранён

**Цель:** Понимать, как размытый вид выглядит для вашей аудитории.

1. При включённом размытии всё текстовое содержимое на страницах Jira отображается как нечитаемое размытие — заголовки, ключи, описания, имена пользователей, значения пользовательских полей становятся нечитаемыми.
2. Изображения (аватары, скриншоты, превью вложений) размыты.
3. **Макет и позиции карточек** страницы остаются неизменными — колонки, дорожки и структура сетки по-прежнему навигируемы.
4. Выключите размытие из того же флажка **правый клик → Blur secret data**, когда закончите презентацию.

## Устранение неполадок

### В контекстном меню нет «Blur secret data»

- Расширение должно быть **включено** для домена Jira. Проверьте `chrome://extensions` или `about:addons` — убедитесь, что Jira Helper включён и имеет доступ к сайту.
- В Chrome запись может быть вложена в подменю **Jira Helper** — ищите имя/значок расширения в контекстном меню, затем разверните его.
- В Firefox запись контекстного меню появляется на верхнем уровне, но только если расширение активно на текущей вкладке.
- Если запись всё ещё отсутствует, **перезагрузите страницу Jira** (F5) и кликните правой кнопкой снова.

### Размытие не применяется при включении

- Элементы страницы, загруженные динамически (AJAX) после включения размытия, могут быть не размыты. **Выключите и снова включите размытие**, чтобы повторно применить эффект к новому содержимому.
- Некоторые кастомизации Jira или плагины Marketplace внедряют содержимое через iframe — содержимое внутри кросс-доменных iframe не может быть размыто расширением.
- Текст SVG и содержимое, отрисованное на canvas, не могут быть размыты через CSS и останутся видимыми даже при включённом размытии.

### Непреднамеренно оставили размытие включённым

- Состояние размытия отображается только галочкой в контекстном меню — постоянного индикатора на странице нет. Если вы вернулись в Jira и текст выглядит размытым, кликните правой кнопкой и снимите отметку **Blur secret data**.
- Выработайте привычку: после презентации сразу снимайте флажок. Состояние сохраняется между перезапусками браузера.
