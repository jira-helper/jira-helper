# Шаблоны комментариев

| | |
|---|---|
| Где настраивается | Настройки доски → вкладка **«Шаблоны комментариев»** или Настройки задачи → вкладка **«Шаблоны комментариев»** |
| Где видно | Страница задачи (тулбар редактора комментариев) |
| Настройки действуют | Только для вас |(`jira_helper_comment_templates`) |

## Цель

Предоставить переиспользуемые шаблоны комментариев, которые вставляются одним кликом из тулбара редактора комментариев, с опциональным автоматическим добавлением наблюдателей.

## Как настроить

### Где найти настройки

Диалог шаблонов комментариев открывается двумя способами:

**Через настройки доски:**

1. Откройте доску Jira и нажмите **«Jira Helper»** рядом с боковой панелью.
2. Выберите вкладку **«Шаблоны комментариев»**.

**Через настройки задачи:**

1. Откройте любую задачу.
2. Нажмите **«Helper»** в тулбаре задачи.
3. Выберите вкладку **«Шаблоны комментариев»**.

Оба способа открывают один и тот же диалог.

### Как настроить

- **Создать шаблон**: нажмите **«Добавить шаблон»**. Задайте **«Название»** (название на кнопке), **«Цвет»** (предустановленный или hex-цвет), **«Текст»** (текст комментария) и опционально **«Наблюдатели»** (пользователи, добавляемые в наблюдатели при вставке шаблона).
- **Редактировать шаблон**: измените название, цвет, текст или наблюдателей существующего шаблона.
- **Переупорядочить шаблоны**: измените порядок кнопок в тулбаре перетаскиванием.
- **Удалить шаблон**: удалите ненужный шаблон.
- **Импорт и экспорт**: используйте **«Импортировать JSON»** и **«Экспортировать шаблоны»** для переноса шаблонов между браузерами.
- **Сбросить настройки**: нажмите **«Сбросить до стандартных»** для восстановления встроенного набора шаблонов.

Нажмите **«Сохранить»**, чтобы сохранить изменения в локальном хранилище браузера.

## Как использовать

- В **редакторе комментариев** на любой странице задачи появляется тулбар с подписанными цветными кнопками шаблонов.
- Нажмите кнопку шаблона, чтобы **вставить** его текст в поле комментария.
- Если в шаблоне настроены наблюдатели, они автоматически добавляются к задаче.
- Кратковременное уведомление подтверждает успех вставки или сообщает об ошибках добавления наблюдателей.
- Нажмите **«Управление шаблонами»** в тулбаре для открытия панели настроек.

## Сценарии использования

- **Стандартные ответы:** Создавайте шаблоны для типовых ответов (например, «Буду разбираться», «Слито в master», «Нужно ревью»).
- **Онбординг:** Делитесь JSON-экспортом с новыми членами команды, чтобы у них были те же шаблоны.
- **Автоматизация наблюдателей:** Включайте наблюдателей в шаблоны для автоматического оповещения нужных людей.
- **Многоязычные команды:** Создавайте шаблоны на разных языках для кросс-командной коммуникации.
