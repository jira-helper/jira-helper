# Гистограмма дорожек (Столбчатая диаграмма)

## Обзор

Гистограмма дорожек добавляет компактную столбчатую диаграмму рядом с заголовком каждой дорожки на доске, показывая, как задачи в этой дорожке распределены по колонкам.

**Зачем:** На досках с большим количеством дорожек (по исполнителю, эпику или пользовательским) трудно с первого взгляда понять, сбалансирована ли работа или формируются ли узкие места. Гистограмма предоставляет мгновенные данные о распределении по каждой дорожке без прокрутки или подсчёта.

**Где:** Появляется автоматически в детальных представлениях доски (`?view=detail`), когда дорожки активны. Не требуется панель настроек или конфигурация — нулевая настройка, нулевое вмешательство администратора.

**Как это работает:**
- Для каждой дорожки диаграмма подсчитывает задачи в каждой колонке
- Столбцы рисуются пропорционально — самый высокий столбец представляет колонку с наибольшим количеством задач
- Серые столбцы: более тёмные столбцы указывают на колонки с задачами, более светлые — заполнители для пустых колонок
- Наведите курсор на столбец, чтобы увидеть подсказку с названием колонки и точным количеством задач
- Обновляется автоматически при изменении доски (перетаскивание, изменение фильтров и т.д.)

**Характер функции:** Гистограмма — это **визуализация только для чтения, без взаимодействия**. Вы не можете нажимать на столбцы для фильтрации, перетаскивать для переупорядочивания или что-либо настраивать. Она исключительно отображает данные о распределении задач для быстрой визуальной оценки.

**Необходимые условия:**
- Дорожки должны быть активны на доске (подходит любая стратегия дорожек)
- Доска должна быть в **детальном представлении** (`?view=detail`)

**Ограничения:** Не отображается в упрощённых или альтернативных представлениях доски (`?view=planning`, `?view=summary`).

## Пользовательские сценарии

### Нулевая настройка (настройки не требуются)

**Цель:** Видеть гистограмму без какой-либо настройки.

- Откройте любую доску с включёнными дорожками в стандартном детальном представлении.
- Гистограмма видна рядом с заголовком каждой дорожки сразу — не требуется никаких настроек доски, переключателей или конфигурации.
- Функция активируется автоматически на досках, где активно `?view=detail` (представление доски по умолчанию).

### Представление доски: столбцы гистограммы, наведение для подсказки с количеством задач

**Цель:** Интерпретировать распределение задач по дорожкам.

1. Посмотрите на строку заголовка дорожки. С правой стороны заголовка отображается горизонтальный ряд тонких вертикальных столбцов — по одному на каждую колонку доски.
2. Сравните высоту столбцов: более высокий столбец означает, что большая доля задач этой дорожки находится в соответствующей колонке.
3. **Наведите курсор** на любой столбец, чтобы увидеть подсказку с:
   - Названием колонки
   - Точным количеством задач в этой колонке для данной дорожки
4. Используйте гистограмму для выявления дисбаланса с первого взгляда:
   - Дорожка, где все задачи собраны в одной колонке, может указывать на узкое место
   - Дорожка с равномерно распределёнными задачами может указывать на сбалансированный поток
5. После перетаскивания карточки в другую колонку или изменения назначений дорожек гистограмма автоматически обновляется, отражая новое распределение.

## Пограничное поведение

- **Пустая дорожка:** Дорожка без задач отображает все столбцы в светло-сером стиле заполнителей — все столбцы отображаются с одинаковой минимальной высотой. Подсказки недоступны, так как нет количества задач.
- **Много колонок:** Доски с большим количеством колонок (15+) могут привести к расширению гистограммы. Столбцы сжимаются, но остаются различимыми. При необходимости прокрутите доску по горизонтали — гистограмма прокручивается вместе с заголовком дорожки.
- **Одна задача:** Дорожка с одной задачей показывает один тёмный столбец (для колонки, содержащей задачу) и все остальные столбцы светло-серыми.

## Технические примечания

- **Триггеры обновления:** Гистограмма пересчитывается и перерисовывается при: перетаскивании карточек, изменении Быстрых фильтров, изменении стратегии дорожек, перезагрузке доски и обновлении JQL-фильтров.
- **Источник данных:** Количество задач извлекается из видимого DOM — гистограмма считывает то, что в данный момент отображается на доске. Она не запрашивает API Jira отдельно.
- **Производительность:** Отрисовка легковесна и не оказывает заметного влияния на доски с менее чем 500 видимыми карточками.

## Устранение неполадок

### Гистограмма не отображается

1. Убедитесь, что вы находитесь в **детальном представлении доски** (`?view=detail`). Гистограмма не отображается в Бэклоге (`?view=detail&layout=list`), Планировании или Сводке.
2. Дорожки должны быть **активны** на доске. Если на доске нет строк дорожек (например, дорожка установлена в «Нет»), гистограмме нечего отображать.
3. Функция автоматическая и не имеет переключателя настроек — если она не отображается, проверьте, что jira-helper установлен и включён.
4. Обновите страницу доски.

### Гистограмма показывает отсутствие столбцов (пустая область)

1. Если на доске есть дорожки, но **нет видимых задач** (пустая доска, все задачи отфильтрованы), гистограмма отображается пустой.
2. Быстрые фильтры могут скрывать все задачи — очистите все фильтры для проверки.

### Подсказка не отображается при наведении

1. Наведение должно быть непосредственно на столбец. Светло-серые столбцы-заполнители для пустых колонок не показывают подсказки.
2. Если столбец очень короткий (очень мало задач в этой колонке), область наведения мала — попробуйте навести курсор на самый верх столбца.
3. Расширения браузера или блокировщики рекламы, мешающие отображению подсказок, могут вызывать проблемы — протестируйте в окне инкогнито.
