# Установка

Jira Helper — это браузерное расширение, которое добавляет WIP-лимиты, диаграмму Ганта и другие полезные функции в Jira. Работает с Jira Server и Jira Data Center. Jira Cloud не поддерживается.

**Приватность:** Никакие данные не передаются третьим лицам. Расширение взаимодействует только с вашим экземпляром Jira через его существующий API. Никакой аналитики, телеметрии, внешних серверов.

## Поддерживаемые браузеры

- **Chrome**, **Edge**, **Yandex** и другие браузеры на Chromium
- **Firefox**
- Safari не поддерживается

## Установка

### Chromium (Chrome, Edge, Yandex и др.)

1. Откройте [страницу в Chrome Web Store](https://chrome.google.com/webstore/detail/jira-helper/egombomekcmpieccamghfgjgnlllgbgdl)
2. Нажмите **Add to Chrome** → **Add Extension**

### Firefox

1. Откройте [страницу дополнений Firefox](https://addons.mozilla.org/ru/firefox/addon/jira-helper-for-ff/)
2. Нажмите **Add to Firefox** → **Add**

## После установки

1. Перезагрузите страницу Jira
2. Откройте любую Scrum- или Kanban-доску
3. На панели инструментов доски появится кнопка **Jira Helper** — квадратная кнопка с надписью «Jira Helper», справа от значка шестерёнки Jira

Также можно открыть **Настройки доски** — в различных секциях появятся кнопки настройки Jira Helper: **Колонки**, **Swimlanes**, **Card layout** и других.
