# FAQ

## General

### Is jira-helper free?

Yes. jira-helper is open source under the ISC license. It's free to use and modify.

### Does it work with Jira Cloud and Jira Server?

Yes. jira-helper works with both Jira Cloud and Jira Server/Data Center.

### Which browsers are supported?

Chrome 88+ and Firefox 58+. The extension uses WebExtensions APIs for broad compatibility.

### Does it send my data anywhere?

No. All data stays in your browser. The extension makes requests to your Jira instance via its existing APIs — no data is sent to third parties.

## Features

### Can I use WIP limits and the Gantt chart together?

Yes. All features are independent and can be combined freely.

### Do I need to configure every feature?

No. Each feature is optional. Some work out of the box (Swimlane Histogram, Flag on Issue Panel), others need configuration in board settings.

## Troubleshooting

### The extension doesn't appear on my board

1. Make sure the extension is installed and pinned
2. Refresh the Jira page
3. Check that you're on a supported board view (not backlog)

### Settings don't save

Check that you have board edit permissions in Jira. Board property settings require editor access.
