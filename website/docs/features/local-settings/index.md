# Local Settings

## Overview

Set the extension's UI language independently of Jira's system language. Adds a **Local Settings** tab to the Jira Helper board settings panel with **Auto** (browser default), **English**, or **Russian**.

---

## User Jobs

### Open Settings → Local Settings tab

**Goal:** Access the language preference panel.

1. Open any board → click **Jira Helper** next to the sidebar.
2. In the board settings popup, navigate to the **Local Settings** tab.

### Choose language: Auto (browser default) / English / Russian

**Goal:** Switch the extension interface to your preferred language.

1. In the **Local Settings** tab, find the **UI language** selector.
2. Choose one of:
   - **Auto** — follows the browser's language setting.
   - **English** — forces the extension UI to English.
   - **Russian** — forces the extension UI to Russian.
3. The change applies immediately — no Save button needed.
4. The setting is stored locally in the browser (personal, not synced to Jira).
5. Only the extension's own labels and messages change; Jira's menus and system language are unaffected.
