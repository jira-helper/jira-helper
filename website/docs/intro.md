# Features Overview

jira-helper adds professional Kanban capabilities to Jira. All features are optional and configurable per board.

## WIP Limits

Control work-in-progress with five types of limits:

- **[Column Group WIP Limits](/docs/features/wip-limits/column-limits)** — shared limits across multiple columns (CONWIP)
- **[Swimlane WIP Limits](/docs/features/wip-limits/swimlane-limits)** — per-swimlane caps
- **[Personal WIP Limits](/docs/features/wip-limits/personal-limits)** — per-person workload balance
- **[Field Value WIP Limits](/docs/features/wip-limits/field-limits)** — capacity allocation by field values
- **[Cell WIP Limits](/docs/features/wip-limits/cell-limits)** — limits on column + swimlane intersections

## Board Visualization

- **[Card Colors](/docs/features/board-visualization/card-colors)** — full card highlighting based on JQL
- **[Swimlane Histogram](/docs/features/board-visualization/swimlane-histogram)** — issue distribution chart per swimlane

## Card Information

- **[Days in Column](/docs/features/card-information/days-in-column)** — how long has the issue been here
- **[Days to Deadline](/docs/features/card-information/days-to-deadline)** — remaining time with color coding
- **[Issue Links Display](/docs/features/card-information/issue-links-display)** — related issues on cards
- **[Issue Condition Checks](/docs/features/card-information/issue-condition-checks)** — custom icon badges via JQL

## Progress & Planning

- **[Sub-tasks Progress](/docs/features/sub-tasks-progress)** — progress bars for subtasks and epics
- **[Gantt Chart](/docs/features/gantt-chart)** — interactive timeline on any issue page

## Analytics

- **[SLA Line](/docs/features/control-chart/sla-line)** — percentile reference on Control Chart
- **[Scale Ruler](/docs/features/control-chart/scale-ruler)** — measurement grid for estimation analysis

## Other

- **[Flag on Issue Panel](/docs/features/flag-issue)** — visual flag indicators
- **[Comment Templates](/docs/features/issue-templates/comment-templates)** — reusable comment templates
- **[Data Blurring](/docs/features/data-blurring)** — hide sensitive data during presentations
- **[Local Settings](/docs/features/local-settings)** — language preferences
