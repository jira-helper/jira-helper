# About

Jira Helper is a browser extension that adds useful features to Jira for task management: WIP limits, visualization, Gantt charts, analytics, and more.

The extension works with **Jira Server** and **Jira Data Center**. Jira Cloud is not supported.

**Supported browsers:** Chrome, Edge, Yandex, and other Chromium-based browsers, as well as Firefox. Safari is not supported.

All features are optional and configured per board.

> **New to Jira Helper?** Start with the [Quick Start guide](/docs/getting-started/quick-start).
