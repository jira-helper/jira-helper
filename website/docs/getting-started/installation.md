# Getting Started

Install jira-helper browser extension and turn Jira into a real Kanban system.

## Installation

### Chrome / Chromium

1. Open the [Chrome Web Store page](https://chrome.google.com/webstore/detail/jira-helper/egmbomekcmpieccamghfgjgnlllgbgdl)
2. Click "Add to Chrome"
3. Click "Add Extension" in the dialog
4. Pin the extension (optional): click the puzzle icon in the toolbar, then the pin icon next to jira-helper

### Firefox

1. Open the [Firefox Add-ons page](https://addons.mozilla.org/firefox/addon/jira-helper/)
2. Click "Add to Firefox"
3. Click "Add" in the dialog

### Requirements

- Chrome 88+ or Firefox 58+
- A Jira instance with board access
- No additional software or accounts required
