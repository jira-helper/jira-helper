# Флаг на задаче

| | |
|---|---|
| Где настраивается | Настройка не требуется |
| Где видно | Страница задачи, панель задачи на доске, результаты поиска |
| Настройки действуют | Отсутствует (нулевая конфигурация) |

## Цель

Визуально выделять задачи с флагом в иерархии и на странице задачи, чтобы легко замечать помеченную флагом работу.

## Как настроить

Настройка не требуется. Фича работает автоматически на страницах задач, в панели деталей задачи на доске и в результатах поиска.

## Как использовать

- В **иерархии задачи** (связанные задачи, подзадачи, дети эпика) строки задач с флагом подсвечиваются жёлтым фоном и показывают иконку флага.
- На **открытой задаче**, если она помечена флагом, иконка флага отображается в области заголовка (рядом с приоритетом/типом) в классическом виде задачи.
- В **боковой панели деталей** на доске связанные элементы с флагом подсвечиваются и помечаются иконкой флага.
- Фича также добавляет переключатель **«свернуть/развернуть»** для правой боковой панели на классической странице задачи, позволяя освободить место по ширине.

## Сценарии использования

- **Быстрая сортировка:** Во время обзора спринта замечать задачи с флагом, требующие немедленного внимания.
- **Осведомлённость о зависимостях:** Видеть, когда связанные или дочерние задачи помечены флагом, что указывает на потенциальные блокировки.
- **Оптимизация пространства:** Сворачивать правую боковую панель, чтобы сосредоточиться на основном содержании задачи.
