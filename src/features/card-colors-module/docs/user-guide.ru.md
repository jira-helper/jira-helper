# Цвета карточек (Card Colors)

| | |
|---|---|
| Где настраивается | Настройки доски → Card Colors |
| Где видно | Доска (детальный вид) |
| Настройки действуют | Для всей команды |(`card-colors`) |

## Цель

Усилить заметность правил **Card Colors** (Jira): применить мягкий фоновый оттенок цвета ко всей карточке задачи, а не только к узкой полоске слева.

## Как настроить

1. Откройте доску и перейдите в **Настройки доски** (шестерёнка).
2. Откройте вкладку **Card Colors** — стандартный экран Jira для настройки JQL-правил цветов.
3. Над таблицей правил найдите переключатель **Fill whole card**.
4. Включите опцию — карточки заливаются цветом целиком; выключите — остаётся только полоска слева (стандартное поведение Jira).

Изменять настройку могут только администраторы доски. Выбор сохраняется как board property вместе с остальными настройками.

## Как использовать

- При включённой опции каждая карточка получает светлую фоновую заливку в цвет, заданный правилом Card Colors. Левая цветная полоска Jira сохраняется.
- **Помеченные** (flagged) карточки и карточки с имеющейся специальной подсветкой (например, предупреждения WIP-лимитов) остаются без изменений, чтобы срочные сигналы не терялись.
- Настройка применяется глобально для всей доски.

## Сценарии использования

1. **Улучшение читаемости** — полная заливка делает цветовую кодировку задач заметнее, особенно на больших досках.
2. **Совместимость с WIP-лимитами** — карточки с предупреждением о превышении WIP-лимита не перекрашиваются, сохраняя приоритет важных сигналов.
3. **Быстрое отключение** — если заливка мешает, администратор отключает её в один клик, возвращая стандартный вид (только полоска).
