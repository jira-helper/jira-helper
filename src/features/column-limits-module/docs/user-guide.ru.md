# Групповые лимиты WIP по колонкам (CONWIP)

| | |
|---|---|
| Где настраивается | Настройки доски → Колонки → «WIP-лимиты на группы колонок»<br/>**или** панель **«Jira Helper»** на доске → вкладка **«WIP-лимиты по колонкам»** |
| Где видно | Доска (детальный вид) |
| Настройки действуют | Для всей команды |

## Цель

Объединить несколько колонок под одним общим лимитом незавершённой работы, чтобы карточки во всех этих колонках учитывались в едином счётчике. Возможна фильтрация по swimlane и типам задач, плюс наглядное выделение групп цветом и предупреждения о превышении.

## Как настроить

### Где найти настройки

Диалог WIP-лимитов по колонкам открывается двумя способами:

**Через Настройки доски:**

1. Откройте **«Настройки доски»** для Scrum- или Kanban-доски.
2. Перейдите на вкладку **«Колонки»**.
3. Нажмите **«WIP-лимиты на группы колонок»**.

**С панели Jira Helper на доске:**

1. На тулбаре доски откройте панель **«Jira Helper»**.
2. Перейдите на вкладку **«WIP-лимиты по колонкам»**.

Оба способа открывают один и тот же диалог.

### Как настроить

В диалоговом окне доступны следующие действия:

- **Создать группу**: перетащите колонку в свободную область и задайте имя группы. Группа получает общий WIP-лимит — задачи во всех колонках группы суммируются в один счётчик.
- **Добавить колонки в группу**: перетащите другие колонки из списка в нужную группу.
- **Задать лимит**: укажите максимальное количество задач, которые могут одновременно находиться в колонках группы.
- **Выбрать цвет**: задайте цвет для заголовков колонок группы — помогает визуально отличать группы на доске. Если цвет не выбран, он подбирается автоматически по имени группы.
- **Выбрать swimlane**: ограничьте действие лимита только определёнными swimlane.
- **Выбрать типы задач**: укажите, какие типы задач (Bug, Task, Story и т.д.) учитываются в лимите группы.

Нажмите **«Сохранить»**, чтобы сохранить настройки для всей команды на этой доске.

## Как использовать

- На доске каждая группа колонок показывает бейдж **текущее / лимит** в заголовке первой колонки группы.
- Заголовки колонок группы стилизуются цветом группы, чтобы группу было легко узнать.
- При превышении лимита затронутые области колонок подсвечиваются **красным**.
- Если в настройках колонок доски включён учёт подзадач, подзадачи тоже учитываются в лимите.

## Сценарии использования

- «Хочу выставить WIP-лимит на всю взятую в работу от To Do до Done.»
- «Хочу выставить WIP-лимиты на разработку (3 колонки) и тестирование (2 колонки).»
