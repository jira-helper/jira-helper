# Персональные лимиты WIP

| | |
|---|---|
| Где настраивается | Настройки доски → Колонки → **«WIP-лимиты на человека»**<br/>**или** панель **«Jira Helper»** на доске → вкладка **«Персональные лимиты»** |
| Где видно | Доска (детальный вид) |
| Настройки действуют | Для всей команды |

## Цель

Ограничить, сколько задач каждый участник команды может иметь в работе на доске. У каждого лимита — свой потолок и область действия (колонки, swimlane, типы задач). На доске отображаются бейджи с аватарами и счётчиками, подсвечиваются перегруженные карточки, а клик по бейджу фильтрует доску для фокусировки на загрузке конкретного человека.

## Как настроить

### Где найти настройки

Диалог персональных лимитов открывается двумя способами:

**Через Настройки доски:**

1. Откройте **«Настройки доски»**.
2. Перейдите на вкладку **«Колонки»**.
3. Нажмите **«WIP-лимиты на человека»**.

**С панели Jira Helper на доске:**

1. На тулбаре доски откройте панель **«Jira Helper»**.
2. Перейдите на вкладку **«Персональные лимиты»**.

Оба способа открывают один и тот же диалог.

### Как настроить

В модальном окне доступны следующие действия:

- **Добавить лимит** — найдите пользователя по имени, задайте **максимальное количество задач** и при необходимости сузьте область по **колонкам**, **swimlane** и **типам задач**.
- **Указать нескольких людей** — можно добавить несколько человек в один лимит и выбрать режим:
  - **Общий (shared)**: лимит действует на всех вместе. Например, лимит 5 на четверых — общий счётчик на команду.
  - **Каждому (per-user)**: каждый получает свой лимит. Например, лимит 5 на четверых — каждый по 5 задач.
- **Редактировать лимит** — нажмите **«Редактировать»** в строке существующего лимита и измените параметры.
- **Удалить лимит** — нажмите **«Удалить»** в строке лимита.
- **Создать несколько лимитов на человека** — можно задать несколько лимитов с разными областями для одного сотрудника. Каждый лимит считается и отображается независимо.

Нажмите **«Сохранить»**, чтобы сохранить, или **«Отмена»**, чтобы отменить.

## Как использовать

- В панели над доской появляются бейджи с аватарами и счётчиком **текущее / лимит** для каждого настроенного сотрудника.
- Цвета бейджей: **зелёный** (ниже лимита), **жёлтый** (на лимите), **красный** (выше лимита).
- При превышении лимита карточки сотрудника, входящие в этот лимит, подсвечиваются **красным**.
- **Клик по бейджу** фильтрует доску — остаются только карточки, соответствующие области этого лимита. Повторный клик снимает фильтр.

## Сценарии использования

- «Хочу на стендапе сразу видеть, кто перегружен.»
- «Хочу ограничить разработчиков тремя задачами в работе, независимо от колонки.»
- «Хочу считать только баги для лимита конкретного сотрудника.»
- «Хочу отфильтровать доску так, чтобы видеть только задачи конкретного участника.»
