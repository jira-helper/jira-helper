# Лимиты WIP по значениям полей

| | |
|---|---|
| Где настраивается | Настройки доски → Card layout → **«WIP-лимиты по полям»** |
| Где видно | Доска (детальный вид) |
| Настройки действуют | Для всей команды |

## Цель

Превратить поле карточки в лимит по принципу ёмкости. Можно считать или суммировать вклад задач по значению поля (или по факту его заполнения), с ограничением области колонками и swimlane. Бейджи над доской показывают соотношение «текущее / лимит», а карточки подсвечиваются при превышении.

## Как настроить

### Где найти настройки

1. Откройте **«Настройки доски»**.
2. Перейдите на вкладку **«Карточка»**.
3. Убедитесь, что нужное поле отображается на карточках доски.
4. Нажмите **«WIP-лимиты по полям»**.

### Как настроить

Для каждого правила доступны следующие параметры:

- **«Поле»** — выберите поле из раскладки карточки доски.
- **«Режим расчёта»** — способ учёта: поле заполнено, точное совпадение, любое из выбранных значений или сумма чисел.
- **«Значения»** — если требует режим, укажите, какие значения учитывать.
- **«Отображаемое имя»** — короткая подпись для бейджа.
- **Лимит** — числовой порог.
- **«Цвет бейджа»** — опционально, задайте цвет для бейджа в панели.
- **«Область по колонкам»** — опционально, сузьте учёт до выбранных колонок.
- **«Область по swimlane»** — опционально, сузьте учёт до выбранных swimlane.

Нажмите **«Сохранить»**, чтобы сохранить, или **«Отмена»**, чтобы отменить.

## Как использовать

- В панели над доской появляется ряд **бейджей** — по одному на каждое настроенное правило — с заданными вами подписями и цветами.
- Цвета показывают статус: **зелёный** (ниже лимита), **жёлтый** (на лимите), **красный** (выше лимита).
- При **превышении** лимита карточки, входящие в счётчик правила, получают **красную подсветку**.
- Счёт обновляется при перемещении карточек и учитывает настройки подсчёта задач на уровне доски Jira.

## Сценарии использования

- «Хочу ограничить количество высокоприоритетных задач в работе по полю Priority.»
- «Хочу суммировать Story Points в колонке для контроля ёмкости.»
- «Хочу посчитать, сколько задач с определённой меткой, и ограничить это количество.»
- «Хочу видеть отдельные бейджи для разных значений полей в верхней части доски.»
