# Отображение связей задач (Issue Links Display)

| | |
|---|---|
| Где настраивается | «Настройки доски» → «Дополнительные элементы карточек» → «Связи задач» |
| Где видно | Доска (детальный вид + бэклог) |
| Настройки действуют | Для всей команды ||

## Цель

Отображать связанные задачи прямо на карточках доски в виде цветных бейджей под заголовком (summary) задачи, чтобы быстро видеть контекст и зависимости.

## Как настроить

### Где найти настройки

1. Откройте доску, затем **«Jira Helper»** → «Настройки доски».
2. Перейдите на вкладку **«Дополнительные элементы карточек»**.

### Как настроить

- **«Включить дополнительные элементы»**: включите главный переключатель **«Включить дополнительные элементы»**.
- **«Выбрать колонки»**: отметьте колонки, в которых должны отображаться бейджи связей, в секции **«Выбор колонок»**.
- **«Показывать связи в бэклоге»**: включите **«Показывать связи в бэклоге»**, чтобы связи отображались и на карточках бэклога.
- **«Добавить конфигурацию связи»**: нажмите **«Добавить конфигурацию связи»** и настройте правило:
  - **«Название»** — название конфигурации (до 20 символов, видно только в настройках).
  - **«Тип связи»** — тип связи (например, «is Parent of», «blocks», «relates to»).
  - **«Уникальные цвета»** — включено: каждой связанной задаче назначается уникальный цвет; выключено: фиксированный цвет для всех.
  - **«Многострочный заголовок»** — включено: длинные названия переносятся на несколько строк.
  - **«Отслеживать все задачи»** — для каких задач анализировать связи (все или по JQL/фильтру).
  - **«Отслеживать все связанные задачи»** — какие связанные задачи показывать (все или по JQL/фильтру).
- **«Добавить несколько конфигураций»**: при необходимости добавьте несколько конфигураций для разных типов связей.

Нажмите **«Сохранить»**, чтобы сохранить настройки для всей команды на этой доске.

## Как использовать

**На доске**:
- Бейджи отображаются вертикально под заголовком карточки в выбранных колонках.
- При клике на бейдж связанная задача открывается в новой вкладке.
- Цвет бейджа зависит от настроек: фиксированный или автоматически сгенерированный.

**В бэклоге**:
- При включённой опции **«Показывать связи в бэклоге»** бейджи отображаются горизонтально (в ряд) в конце карточки.
- Настройки колонок в бэклоге не учитываются — связи показываются для всех задач.

## Сценарии использования

1. **Показ родительских задач** — настроить связь «is Parent of», видеть проект или эпик, к которому относится задача.
2. **Фильтрация по типу задачи** — показывать только связанные задачи типа «Project» с фильтром по JQL: `issueType = Project`.
3. **Фильтрация по статусу** — показывать только незавершённые связанные задачи: `status != Done`.
4. **Комбинированный фильтр** — `(issueType = Project AND status != Done) OR (issueType = Objective AND labels = "Business")`.
