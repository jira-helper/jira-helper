# Дни до дедлайна (Days to Deadline)

| | |
|---|---|
| Где настраивается | «Настройки доски» → «Дополнительные элементы карточек» → «Бейдж «Дней до дедлайна»» |
| Где видно | Доска (детальный вид) |
| Настройки действуют | Для всей команды ||

## Цель

Показать, сколько дней осталось до дедлайна задачи, с цветовой индикацией приближающихся и пропущенных сроков. Бейдж отмечен эмодзи ⏰ для визуального отличия от других бейджей на карточке.

## Как настроить

### Где найти настройки

1. Откройте доску, затем **«Jira Helper»** → «Настройки доски».
2. Перейдите на вкладку **«Дополнительные элементы карточек»**.

### Как настроить

- **«Включить дополнительные элементы»**: включите главный переключатель **«Включить дополнительные элементы»**.
- **«Выбрать колонки»**: отметьте колонки, в которых должен отображаться бейдж, в секции **«Выбор колонок»**.
- **«Включить бейдж»**: включите **«Показывать бейдж с днями до дедлайна»** в секции **«Бейдж «Дней до дедлайна»»**.
- **«Выбрать поле дедлайна»**: в выпадающем списке выберите поле, содержащее дату дедлайна (типы `date`, `datetime`, `string`).
- **«Выбрать режим отображения»**:
  - **«Всегда»** — бейдж показывается всегда (если есть дедлайн).
  - **«Менее X дней или просрочено»** — показывается, если осталось ≤ X дней или дедлайн просрочен.
  - **«Только просрочено»** — показывается только для просроченных задач.
- **«Задать порог «Warning»»** (жёлтый): бейдж станет жёлтым, если осталось не больше этого числа дней.

Нажмите **«Сохранить»**, чтобы сохранить настройки для всей команды на этой доске.

## Как использовать

- Бейдж отображается в конце карточки (после бейджа «Дни в колонке», если он включён).
- **Формат текста**:
  - Просрочено: `⏰ Просрочено на X дн.`
  - Сегодня: `⏰ Сегодня!` (красный текст на жёлтом фоне)
  - Завтра: `⏰ Завтра` (жёлтый фон)
  - Осталось дней: `⏰ X дн.`
- **Цвета**:
  - Красный — дедлайн просрочен (всегда).
  - Жёлтый — сегодня (0 дней) или завтра (1 день), либо осталось ≤ warning threshold.
  - Синий — остальные случаи.
- В режиме **«Только просрочено»** жёлтый порог не применяется.
- В бэклоге бейдж не отображается.

## Сценарии использования

1. **Отслеживание всех дедлайнов** — режим «Всегда» показывает бейдж для всех задач с установленным дедлайном.
2. **Фокус на приближающихся сроках** — режим «Менее 5 дней или просрочено» показывает только те задачи, до дедлайна которых осталось 5 или меньше дней.
3. **Работа только с просрочкой** — режим «Только просрочено» для отслеживания исключительно проблемных задач.
