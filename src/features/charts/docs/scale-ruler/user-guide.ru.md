# Измерительная сетка (Scale Ruler)

| | |
|---|---|
| Где настраивается | Reports → Control Chart → чекбокс Grid и выбор пресета в опциях диаграммы |
| Где видно | Reports (Control Chart) |
| Хранилище | Отсутствует (только на сессию) |

## Цель

Наложить на Control Chart перетаскиваемую и масштабируемую измерительную сетку для считывания lead/cycle time относительно шагов в духе story points (пресеты Fibonacci или Linear), упрощая оценку дней по позиции на диаграмме.

## Как настроить

1. Откройте **Reports** для доски и выберите **Control Chart**.
2. В колонке опций диаграммы найдите чекбокс **Grid** и выберите пресет из выпадающего списка.
3. Выберите пресет: **Fibonacci** (1,2,3,5; 1,2,3,5,8; 1,2,3,5,8,13) или **Linear** (2-10 шагов).
4. Оверлей сетки появляется на диаграмме. Перетаскивайте его для позиционирования и используйте маркер изменения размера (правый нижний угол) для подгонки.

Состояние сетки действительно только на сессию и сбрасывается при перезагрузке страницы. Функция не поддерживается в Firefox.

## Как использовать

- Внутри перетаскиваемого оверлея отображаются горизонтальные направляющие линии с подписями story points (SP) и соответствующими днями.
- Каждая линия представляет шаг выбранной последовательности, помогая оценить, сколько дней соответствует данному значению SP.
- Перетаскивайте оверлей для выравнивания линий сетки с точками данных диаграммы.
- Изменяйте размер оверлея для охвата разных областей диаграммы.
- Отключите сетку, сняв чекбокс.

## Сценарии использования

- **Калибровка story points:** Совместить шаги Fibonacci с диаграммой для калибровки командных оценок относительно фактического cycle time.
- **Быстрая оценка:** Использовать линейные шаги для оценки cycle time задач разного размера.
- **Помощь в презентациях:** Использовать сетку при командных обсуждениях для визуализации распределений времени.
- **Анализ процесса:** Накладывать разные пресеты для поиска наилучшего соответствия паттернам поставки команды.

## Устранение неполадок

- **Сетка не отображается:** Функция не поддерживается в Firefox. В Chrome убедитесь, что чекбокс отмечен после загрузки диаграммы.
- **Линии сетки смещены:** Перетащите оверлей для изменения позиции. Линии пересчитываются на основе позиции и размера оверлея.
- **Пресет не меняется:** Выберите другой пресет из выпадающего списка; линии обновляются немедленно.
- **Маркер размера не реагирует:** Маркер находится в правом нижнем углу оверлея. Убедитесь, что оверлей видим, и потяните за угол.
