# Размытие данных

| | |
|---|---|
| Где настраивается | Контекстное меню (правый клик) → «Размыть секретные данные» (переключатель-чекбокс) |
| Где видно | Все страницы Jira (доски, задачи, поиск, бэклог, настройки) |
| Настройки действуют | Только для вас |(`blurSensitive`) |

## Цель

Скрыть читаемый текст и размыть изображения по всему Jira для демонстрации экрана, презентаций или показов — вёрстка остаётся на месте, содержимое становится нечитаемым.

## Как настроить

### Где найти настройки

1. На любой странице Jira откройте **контекстное меню** страницы (правый клик или меню вкладки браузера).
2. Найдите пункт расширения **«Размыть секретные данные»**.

### Как настроить

- **«Включить или выключить размытие»**: переключите пункт **«Размыть секретные данные»** — отмечено = размытие включено, снято = выключено.

Изменение применяется сразу, отдельная кнопка сохранения не требуется. Настройка сохраняется между перезагрузками страниц и новыми вкладками.

## Как использовать

- При включении текст на страницах Jira заменяется эффектом «смазывания» — содержимое не прочитать, но вёрстка не меняется.
- Изображения размываются для предотвращения раскрытия визуальной информации.
- Эффект применяется к доскам, бэклогам, страницам задач, результатам поиска и части экранов настроек.
- Отключите размытие тем же пунктом контекстного меню, когда закончите презентацию или показ экрана.

## Сценарии использования

- **Демонстрация экрана:** Показывать доску Jira на встречах, не раскрывая чувствительные заголовки, ключи и описания задач.
- **Презентации:** Представлять рабочий процесс или информацию о процессе, сохраняя конфиденциальность деталей задач.
- **Запись демо:** Записывать демо продукта с реальными данными, но без раскрытия содержимого.
- **Публичные показы:** Использовать Jira на проекторе или общем экране без ущерба для конфиденциальности данных.
