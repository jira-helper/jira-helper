# Локальные настройки

| | |
|---|---|
| Где настраивается | Доска → Jira Helper → вкладка Local Settings |
| Где видно | Все страницы Jira (только интерфейс расширения) |
| Хранилище | localStorage |

## Цель

Задать язык, на котором расширение Jira Helper показывает свои подписи и сообщения в вашем браузере.

## Как настроить

1. Откройте доску и нажмите **Jira Helper** рядом с боковой панелью.
2. Откройте вкладку **Local Settings**.
3. В **UI language** выберите один из вариантов:
   - **Auto** — следовать языковым настройкам браузера.
   - **English** — принудительно английский для интерфейса расширения.
   - **Russian** — принудительно русский для интерфейса расширения.

Изменение применяется сразу после выбора. Отдельная кнопка сохранения не требуется. Настройка персональная и сохраняется локально в браузере — она не затрагивает язык самой Jira.

## Как использовать

- После выбора языка все элементы интерфейса расширения (всплывающие окна настроек, подписи функций, подсказки) отображаются на выбранном языке.
- При выборе **Auto** расширение использует предпочитаемый язык браузера, если он поддерживается, с резервным английским.
- Меню и системный язык Jira не изменяются.

## Сценарии использования

- **Носители других языков:** Переключить интерфейс расширения на русский для удобства настройки.
- **Смешанные команды:** Каждый член команды устанавливает свой язык независимо — без влияния на других.
- **Несовпадение языка браузера:** Если браузер на одном языке, а расширение хочется на другом, выберите явно.

## Устранение неполадок

- **Язык не меняется:** Настройка применяется сразу. Если нет, попробуйте закрыть и снова открыть всплывающее окно настроек.
- **Auto не совпадает с языком браузера:** Расширение поддерживает английский и русский. Если браузер настроен на другой язык, Auto использует английский.
- **Настройка не сохраняется:** Настройка хранится в localStorage. Если очистить данные браузера, настройка будет потеряна.
