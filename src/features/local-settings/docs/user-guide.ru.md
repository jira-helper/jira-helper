# Локальные настройки

| | |
|---|---|
| Где настраивается | Доска → «Jira Helper» → вкладка «Локальные настройки» |
| Где видно | Все страницы Jira (только интерфейс расширения) |
| Настройки действуют | Только для вас ||

## Цель

Задать язык, на котором расширение Jira Helper показывает свои подписи и сообщения в вашем браузере.

## Как настроить

### Где найти настройки

1. Откройте доску Jira и нажмите **«Jira Helper»** рядом с боковой панелью.
2. Откройте вкладку **«Локальные настройки»**.

### Как настроить

- **«Язык интерфейса»**: выберите один из вариантов:
  - **«Авто»** — следовать языковым настройкам браузера.
  - **«Английский»** — принудительно английский для интерфейса расширения.
  - **«Русский»** — принудительно русский для интерфейса расширения.

Изменение применяется сразу после выбора, отдельная кнопка сохранения не требуется. Настройка сохраняется локально в браузере.

## Как использовать

- После выбора языка все элементы интерфейса расширения (всплывающие окна настроек, подписи функций, подсказки) отображаются на выбранном языке.
- При выборе **«Авто»** расширение использует предпочитаемый язык браузера, если он поддерживается, с резервным английским.
- Меню и системный язык Jira не изменяются.

## Сценарии использования

- **Носители других языков:** Переключить интерфейс расширения на русский для удобства настройки.
- **Смешанные команды:** Каждый член команды устанавливает свой язык независимо — без влияния на других.
- **Несовпадение языка браузера:** Если браузер на одном языке, а расширение хочется на другом, выберите явно.
