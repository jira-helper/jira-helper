- В режиме "все свимлайны" включен только чекбокс **All swimlanes**. Список отдельных свимлайнов скрыт.
- В режиме ручного выбора чекбокс **All swimlanes** выключен. Показывается полный список свимлайнов с индивидуальными галочками.
- Переключение симметричное: из "все" в ручной список и обратно в "все" без промежуточного состояния, где один и тот же выбор дублируется двумя UI-элементами.
