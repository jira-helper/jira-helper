# Request: Фиксы после ручной проверки column-wip-inline

**Дата**: 2026-04-06
**Источник**: ручная проверка пользователем

## Проблема 1: При удалении колонки из группы не стирается старая шапка

При добавлении колонки в группу — на борде сразу всё обновляется корректно.
При убирании колонки из группы — шапка перерисовывается, но старая шапка (стили/индикаторы) НЕ стирается. Остаётся артефакт от предыдущего состояния.

## Проблема 2: UI кнопок Save/Cancel в модалке

Сейчас в модалке панели Jira Helper:
- Слева внизу — кнопки сохранения и отмены от старого решения (от таба)
- Справа внизу — кнопки от модалки текущей (внешней)
- Визуально не разделены

Требуется:
1. **Кнопки таба — справа**, выровнены с кнопками модалки
2. **Кнопки внешней модалки отделены чертой** (это footer модалки)
3. **Тексты кнопок более явные**: "Сохранить конфигурацию" и "Отменить изменения"
4. **При отмене** — восстанавливаем состояние контента таба
5. **При сохранении** — сохраняем и перерисовываем борд
