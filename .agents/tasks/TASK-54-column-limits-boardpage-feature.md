## Описание

Создать исчерпывающий feature файл для BoardPage column-limits, описывающий все сценарии отображения и поведения WIP лимитов групп колонок на доске.

## Файлы

```
src/column-limits/BoardPage/
```

## Что сделать

Создать feature файл со сценариями:

### DISPLAY - Отображение лимитов

- SC-DISPLAY-1: Показать бейдж X/Y на первой колонке группы
- SC-DISPLAY-2: Бейдж обновляется при изменении количества задач
- SC-DISPLAY-3: Колонки группы имеют общий цвет заголовка
- SC-DISPLAY-4: Заголовки групп имеют скруглённые углы на краях

### LIMIT EXCEEDED - Превышение лимитов

- SC-EXCEED-1: Красный фон при превышении лимита группы
- SC-EXCEED-2: Нормальный фон когда в пределах лимита
- SC-EXCEED-3: Граничный случай - ровно на лимите

### SWIMLANE FILTER - Фильтрация свимлейнов

- SC-SWIM-1: Игнорировать задачи в исключённых свимлейнах
- SC-SWIM-2: Подсветка учитывает только активные свимлейны

### ISSUE TYPE FILTER - Фильтрация типов задач

- SC-ISSUE-1: Считать только указанные типы задач
- SC-ISSUE-2: Пустой фильтр = все типы задач

### MULTIPLE GROUPS - Несколько групп

- SC-MULTI-1: Каждая группа имеет свой бейдж
- SC-MULTI-2: Группы могут иметь разные цвета
- SC-MULTI-3: Одна группа превышена, другая нет

### DOM CHANGES - Реакция на изменения

- SC-DOM-1: Пересчёт при перемещении задачи между колонками
- SC-DOM-2: Пересчёт при добавлении новой задачи

## Критерии приёмки

- [x] Сценарии имеют семантические ID (SC-DISPLAY-1, SC-EXCEED-1, ...)

- [x] Покрыты основные и edge cases
