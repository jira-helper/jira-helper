# Request: Сделать ссылки на эпики кликабельными

**Дата**: 2026-04-29
**Slug**: `clickable-epic-links`
**Тип**: UI / bugfix
**Has UI**: yes
**Scope**: medium

## Описание запроса

Issue #20: в jira-helper отображаются ссылки на эпики, но они не кликабельные. Нужно сделать кликабельными:

1. Встроенный бейдж Jira для поля `Epic Link` на карточке задачи.
2. Ссылки на эпики в секции issue links, которую отображает jira-helper.

Ссылка должна вести на `{jiraBaseUrl}/browse/{EPIC_KEY}` и иметь визуальный индикатор кликабельности.

## Уточнения

Контекста из issue достаточно для реализации без дополнительного interview:

- Основной сценарий: пользователь видит epic badge или issue link на карточке и кликом открывает задачу-эпик.
- Базовый URL берется из `window.location.origin`.
- Встроенный Epic Link нужно обрабатывать на Jira board/backlog карточках, где Jira рендерит extra field как текст.
